│                                 │

└─────────────────────────────────┘

Animation sequence:
1. "Connecting to Nomba..." (1s)
2. "Transferring to savings account..." (2s)
3. "✓ ₦[amount] saved successfully!" (confetti)

After payment:
- Deduct from parent wallet balance
- Add to student savings balance
- Update savings progress bar instantly
- Add to savings history:
  { date, type: "Deposit", amount, 
    balanceAfter, reference: "NMB/SAV/2026/[8digits]" }
- Show new savings goal progress:
  "₦35,000 of ₦45,000 saved — 78% ✓"

2. SAVINGS GOAL REACHED ANIMATION:

When savings balance reaches or exceeds fee amount:
- Trigger confetti animation
- Show green banner:
  "🎉 Savings Goal Reached!"
  "Emeka's First Term fee is fully covered!"
  "Admin can now deduct fees from savings."
- Update student record in localStorage

3. ADMIN DEDUCTS FEE FROM SAVINGS:

On Admin → Student Profile → Savings tab:
"Deduct for Fees" button shows:

┌─────────────────────────────────┐
│  📚 DEDUCT FEE FROM SAVINGS     │
│                                 │
│  Student: Emeka Okonkwo         │
│  Term: First Term 2026/2027     │
│  Fee Amount: ₦45,000.00         │
│  Savings Balance: ₦35,000.00    │
│                                 │
│  ⚠ Savings covers 78% of fee   │
│  Remaining after deduction: ₦0  │
│  Outstanding balance: ₦10,000   │
│                                 │
│  [Cancel]  [Deduct ₦35,000]     │
└─────────────────────────────────┘

Animation sequence:
1. "Processing deduction..." (2s)
2. "✓ ₦35,000 deducted from savings!"
3. Fee status updates to "Partial" (yellow badge)
4. Outstanding balance shows: ₦10,000 remaining

Savings history records:
{ date, type: "Fee Deduction", 
  amount: -35000, balanceAfter: 0,
  description: "First Term 2026/2027 fee payment",
  reference: "NMB/DED/2026/[8digits]" }

If savings fully covers fee:
- Fee status → PAID (green badge)
- Confetti animation
- "✓ Fee fully paid from savings!"

6. LOW SAVINGS WARNING:

On Parent Dashboard if savings < 25% of fee:
Show pulsing red warning card:
"⚠️ Low Savings Alert"
"Emeka's savings (₦5,000) won't cover the 
upcoming fee of ₦45,000 due on 15 Sep 2026."
"You need ₦40,000 more."
[Top Up Now] button → opens top up modal directly

On Parent Dashboard if savings between 25-75%:
Show yellow warning card:
"📊 Keep Saving!"
"₦25,000 saved. ₦20,000 more to cover full fee."
[Top Up Savings] button

On Parent Dashboard if savings >= 100% of fee:
Show green card:
"✅ You're all set!"
"Emeka's fee is fully covered by savings."
"Admin will process the deduction before term starts."

8. INSTALLMENT SAVINGS PLAN:

On Parent Pay Fees tab add:
"Save in Installments" button (different from Pay in Installments)

This lets parent set up automatic monthly savings:
- Select amount to save per month: ₦5,000 / ₦10,000 / Custom
- Select start date
- Shows: "You'll reach your ₦45,000 goal in [X] months"
- Calls createDirectDebitMandate() for automatic monthly savings
- On success: "✓ Auto-savings set up! ₦[amount] saved monthly"
- Shows upcoming savings dates on Installments tab
